```bash
  const firebaseConfig = {
    apiKey: "YOUR_API",
    authDomain: "YOUR_DOMAIN",

    databaseURL: "YOUR_DATABASE_URL",
    storageBucket: "STORAGE_BUCKET",
    messagingSenderId: "MSG_ID",
    appId: "APP_ID",
    measurementId: "MEASUREMENT_ID"
};
```
Run the index.html

```bash
  index.html
```
